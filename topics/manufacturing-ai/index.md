# Manufacturing AI

제조 AI는 기술 자체보다 현장 제약과 적용 방식이 더 중요합니다. 이 섹션은 비전공 사용자, 로컬 환경, 운영 제약을 포함해 현실적인 관점에서 문서를 쌓습니다.

## 핵심 관점

- 인터넷 차단 환경과 로컬 운영
- Windows 중심 사용자 환경
- 품질 예측 같은 좁고 선명한 문제 정의
