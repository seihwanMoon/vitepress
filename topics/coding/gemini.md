# Gemini

## 한 줄 소개

Gemini는 단순 코드 생성보다 검색 보조와 멀티모달 맥락 이해가 필요한 상황에서 비교 관찰할 가치가 있는 도구입니다.

## 보는 포인트

- 검색과 정보 수집
- 문서 요약
- 멀티모달 입력 활용 가능성
