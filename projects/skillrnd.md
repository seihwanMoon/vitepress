# skillrnd

## 한 줄 소개

## 문제/목적

여러 agent 생태계의 skill과 활용 패턴은 정보가 흩어져 있어 탐색이 어렵습니다. 이 프로젝트는 설명, 분류, Mermaid, 원문 링크, 활용 레시피를 일관된 형식으로 정리해 탐색성과 재사용성을 높이려는 목적을 가집니다.

## 주요 기능

- VitePress 기반의 출판형 정적 문서 사이트
- 블로그와 주제별 아카이브를 함께 운영하는 정보 구조
- `content:new`, `content:remove` 같은 콘텐츠 운영 명령
- 이후 Codex 기반 자동 수집기와 문서 생성기로 확장 가능한 구조
