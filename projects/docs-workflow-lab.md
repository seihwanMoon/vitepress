## 문제/목적

레포가 많아질수록 프로젝트 소개 형식이 들쭉날쭉해지기 쉽습니다. 공통 섹션과 메타데이터를 미리 정해두면 이후 자동화나 재구성도 쉬워집니다.

## 주요 기능

- 프로젝트 문서 템플릿 초안
- 공통 frontmatter 실험
- 아카이브 문서와 프로젝트 문서의 역할 분리
