# aistudio

## 한 줄 소개

엑셀 파일 하나로 불량예측 AI를 만들고 로컬 PC에서 운영까지 할 수 있도록 설계한 중소제조기업용 AutoML + MLOps 도구입니다.

## 문제/목적

제조 현장에서는 인터넷이 차단된 Windows PC 환경과 비전공 사용자라는 제약이 강합니다. 이 프로젝트는 그런 조건에서도 엑셀 기반 데이터만으로 불량 예측 모델을 만들고 운영할 수 있는 노코드에 가까운 도구를 목표로 합니다.

## 주요 기능

- 로컬 AutoML + MLOps 플랫폼 설계
- `README`, `PROGRESS`, `ARCHITECTURE`, `SETUP`, `PHASE1~3`로 나뉜 문서 중심 개발 구조
- MVP, MLOps, Realtime 단계로 이어지는 명확한 로드맵
- 제조 품질 예측이라는 좁고 현실적인 문제에 집중한 제품 정의

## 기술 스택

- AutoML
- MLOps
- Windows 로컬 환경
