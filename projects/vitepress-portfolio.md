# VitePress 포트폴리오

## 한 줄 소개

GitHub 프로젝트를 문서 중심으로 정리하고 `docs.moonworld.uk`로 배포하기 위한 실제 운영 포트폴리오입니다.

## 문제/목적

프로젝트 링크만 나열하는 포트폴리오는 구현 맥락과 의사결정 배경이 드러나지 않습니다. 이 템플릿은 레포지토리별 설명 문서를 통해 작업 의도와 구현 포인트를 함께 정리하는 것을 목표로 합니다.

## 주요 기능

- 홈, 프로젝트 목록, 운영 아카이브의 3축 구조
- VitePress 기본 검색과 페이지별 메타데이터 구성
- Cloudflare Pages Git 연동 배포와 커스텀 도메인 연결
- `robots.txt`, `manifest`, `sitemap.xml` 기반의 기본 공개 설정

## 기술 스택

- VitePress
- Markdown
- Cloudflare Pages
- Custom Domain
