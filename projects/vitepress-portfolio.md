# VitePress 포트폴리오

## 한 줄 소개

GitHub 프로젝트를 문서 중심으로 정리하고 정적 사이트로 배포하기 위한 기본 포트폴리오 템플릿입니다.

## 문제/목적

프로젝트 링크만 나열하는 포트폴리오는 구현 맥락과 의사결정 배경이 드러나지 않습니다. 이 템플릿은 레포지토리별 설명 문서를 통해 작업 의도와 구현 포인트를 함께 정리하는 것을 목표로 합니다.

## 주요 기능

- 홈, 프로젝트 목록, 문서 아카이브의 3축 구조
- 프로젝트별 공통 frontmatter와 문서 템플릿
- VitePress 기본 검색과 사이드바 구성
- Cloudflare Pages Git 연동 배포를 전제로 한 정적 빌드

## 기술 스택

- VitePress
- Markdown
- Cloudflare Pages
